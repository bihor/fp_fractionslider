Change the path to this template via TypoScript-constants. Example: plugin.tx_fpfractionslider_pi1.view { templateRootPath = fileadmin/bsdist/theme/tmpl/fractionslider/Templates/ partialRootPath = fileadmin/bsdist/theme/tmpl/fractionslider/Partials/ layoutRootPath = fileadmin/bsdist/theme/tmpl/fractionslider/Layouts/ } Now you can customise this template.
